Para comenzar a jugar, se debe compilar el archivo CGame.c con gcc.
-Los archivos .in, son matrices para jugar, el archivo Entrada2AinVal.in es una matriz invalida, que sirve para ver
que la IA cuando no encuentra camino, avisa.
-exitArt.art e inst.art son archivos creados para el juego con el fin de agilizar los diseños dentro de la terminal, 
con la funcion que está dentro del header title.h
-scrClean.h es un header que permite limpiar la consola por cada cambio.


Luego, se generan los archivos de salida respectivos .out, los cuales tambien se explican dentro del video adjunto.
